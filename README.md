-----------Controls--------------

WASD - Movement
Shift - Sprint
Spacebar - Jump
Left Mouse Button - Attack
Right Mouse Button - Fire grappling hook (Click/Hold-Release for different functionality)
Aim - Mouse
Open/Close skill-tree - G
Interact - E
Pause Menu - ESC
Quick-Save - F1
Quick-Load - F5
God Mode + Increase XP = K
